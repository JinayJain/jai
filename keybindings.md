| Mode | Shortcut   | Action                           |
| ---- | ---------- | -------------------------------- |
| Any  | Ctrl+q     | quit                             |
| Any  | Ctrl+s     | save file                        |
| Edit | h, j, k, l | move cursor                      |
| Edit | d          | delete line                      |
| Edit | o          | insert line below and write mode |
| Edit | i          | write mode                       |
